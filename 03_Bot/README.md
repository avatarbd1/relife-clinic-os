# Bot

## Purpose

This folder contains Telegram bot implementation and related files:
- Bot source code and handlers
- Command definitions and responses
- Bot configuration and environment setup
- User interaction flows
- Bot testing and debugging files
- Integration with clinic management system

## Contents

- `handlers/` - Command and message handlers
- `commands/` - Command implementations
- `keyboards/` - Button and keyboard layouts
- `services/` - Bot service integrations
- `config/` - Bot configuration files
- `tests/` - Bot testing files
