## ⚠️ গুরুত্বপূর্ণ সতর্কতা (Claude-2, 2026-07-23)

### ১. GitHub fetch/browsing ব্যর্থ হলে অনুমান কোরো না
তোমার browsing/fetch tool GitHub থেকে ফাইল আনতে ব্যর্থ হতে পারে (ক্যাশিং, permission, network সীমাবদ্ধতার কারণে) — এটা প্রমাণ না যে ফাইল push হয়নি। এমন হলে ব্যবহারকারীকে বলো টার্মিনাল থেকে cat করে ফাইলের কনটেন্ট paste করে দিতে, নিজে অনুমান করে এগিয়ো না।

### ২. bot.py দুইটা কপি আছে — সাবধান, ভুল ফাইলে প্যাচ দিও না
এই রিপোতে দুইটা ভিন্ন bot.py আছে যা একে অপরের সাথে সিঙ্ক নয়:
- bot.py (root) — লাইভ/সক্রিয় ফাইল, আসল পরিবর্তন এখানেই হয়।
- 03_Bot/bot.py — পুরনো/stale কপি (এতে treat_manual-এর মতো নতুন ফাংশন নেই)।

কাজ শুরুর আগে যাচাই করো কোনটা আসল — ls -la দিয়ে সাইজ/তারিখ তুলনা করো, diff bot.py 03_Bot/bot.py দিয়ে ফারাক দেখো, এবং grep দিয়ে তোমার relevant ফাংশন নাম উভয় ফাইলে খোঁজো। সন্দেহ থাকলে ব্যবহারকারীকে জিজ্ঞাসা করো, অনুমান করে patch দিও না। একই নিয়ম (exact-string patch, পুরো ওভাররাইট নয়) root bot.py-এর জন্যও প্রযোজ্য।
